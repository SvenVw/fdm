# `fdm-app`

The `fdm-app` is a comprehensive web application built with [Remix](https://remix.run/) that serves as the primary user interface for the Farm Data Model (FDM). It provides a user-friendly platform for farmers, advisors, and researchers to manage and visualize agricultural data.

## Key Features

- **Interactive Maps**: Visualize farm layouts, field boundaries, and sensor data on a dynamic map.
- **Data Management**: Easily input, update, and track data for farms, fields, crops, and treatments.
- **User Authentication**: Secure login and user management to protect sensitive farm data.
- **Component-Based Architecture**: Built with reusable React components for a consistent and maintainable user interface.

For more detailed information about the project structure, setup, and contribution guidelines, please refer to the [package README](../..//packages/fdm-app/README.md).
